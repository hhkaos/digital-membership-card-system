Verifier User Manual

Who this guide is for

This guide is for AMPA-friendly businesses that need to verify the membership cards of AMPA members.

What Verifier does

When scanning a QR code, the verifier checks:

  • The validity date of the QR code
  • The expiry date
  • That the code was generated by AMPA Novaschool Almería
  • Whether the person is still an AMPA member or has been removed for any reason

Understanding the states

Valid membership

  • Green state with the member's name
  • Shows the validity date
  • Confirms the person is still an AMPA member

Revoked membership

  • The person is no longer part of the AMPA
  • The card/member should not be accepted
  • A contact link is provided for support

Invalid membership

Possible reasons:

  • Expired card
  • Incorrect QR code

You can open Show Technical Details for support diagnostics.

Manual de Usuario - Verificador

Para quién es esta guía

Esta guía es para empresas amigas del AMPA que necesitan verificar los carnets de personas asociadas al AMPA.

Qué hace Verificador

Al escanear un QR, el verificador comprueba:

  • La fecha validez del código QR
  • La fecha de caducidad
  • Que el código ha sido generado por el AMPA Novaschool Almería
  • Y si la persona sigue siendo socia del AMPA o se le ha dado de baja por algún otro motivo

Entender los estados

Carnet en vigor

  • Estado verde con nombre del socio
  • Muestra fecha de validez
  • Confirma que la persona sigue asociada en el AMPA

Carnet dado de baja

  • Dicha persona ya no forma parte del AMPA
  • La tarjeta/socio no debe aceptarse
  • Se incluye enlace de contacto para soporte

Carnet inválido

Posibles motivos:

  • Carnet caducado
  • Código QR incorrecto

Puedes abrir Mostrar detalles técnicos para diagnóstico de soporte.
